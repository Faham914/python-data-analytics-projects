## COVID-19 Trend Analysis & Exploratory Data Analysis

### Project Overview
This project performs exploratory data analysis on COVID-19 data to understand trends over time. The analysis focuses on identifying patterns in confirmed cases, recoveries, and fatalities using time-based visualizations.

---

### Objectives
- Analyze COVID-19 trends over time
- Compare key metrics such as confirmed cases and deaths
- Visualize temporal patterns to support insights

---

### Tools Used
- Python
- pandas
- matplotlib
- seaborn

---

### Key Analysis Performed
- Time-series trend analysis
- Comparative metric visualization
- Distribution and pattern analysis
- Insight-driven interpretation of results

---

### Dataset
- COVID-19 dataset sourced from publicly available data
- Raw data files are not included in this repository

---

### Outcome
The analysis highlights clear temporal trends and patterns, demonstrating the ability to derive meaningful insights from real-world time-series data.
