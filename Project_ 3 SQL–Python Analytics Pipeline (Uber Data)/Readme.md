### Project Overview
This project demonstrates an end-to-end analytics pipeline by integrating SQL Server with Python. Data is extracted from a relational database using SQL queries and analyzed in Python to generate operational insights.

---

### Objectives
- Connect Python to SQL Server
- Execute SQL queries directly from Python
- Perform analytical processing using pandas
- Visualize and export business insights

---

### Tools Used
- Python
- pandas
- matplotlib
- seaborn
- pyodbc
- SQL Server

---

### Workflow
- Established SQL Server connection using Python
- Extracted aggregated operational data using SQL
- Performed KPI analysis in pandas
- Visualized trends such as revenue distribution and cancellations
- Exported analytical summaries for reporting

---

### Dataset
- Uber operational data stored in SQL Server
- Raw database files are intentionally not included

---

### Outcome
This project demonstrates real-world analyst workflows, showcasing the effective use of SQL for data extraction and Python for analysis and visualization.
